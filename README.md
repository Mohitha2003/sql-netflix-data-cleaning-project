# Netflix Data Cleaning Using SQL

## Project Overview
This project demonstrates how to clean real-world data using SQL.

The Netflix Movies and TV Shows dataset was cleaned using MySQL by handling missing values, duplicates, and inconsistent data types.

## Tools Used
- MySQL
- MySQL Workbench
- SQL
- GitHub

## Dataset
Netflix Movies and TV Shows dataset.

## Data Cleaning Steps
1. Created a copy of the dataset
2. Checked duplicates using ROW_NUMBER()
3. Identified NULL and blank values
4. Replaced missing values with "Unknown"
5. Converted date format using STR_TO_DATE()
6. Changed column data types
7. Validated cleaned dataset
